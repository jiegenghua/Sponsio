# Code of Conduct

The Sponsio project has adopted the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
as the code of conduct for our community.

The full text — including expected behavior, unacceptable behavior, scope,
and enforcement guidelines — is available at the canonical URL above and
applies in full to all project spaces (the GitHub repository, issue
tracker, pull request reviews, discussions, and any official Sponsio
communication channel).

## Reporting

If you experience or witness behavior that violates the Code of Conduct,
please report it by emailing **folio.ai.initial@gmail.com**. All reports
will be reviewed and investigated promptly and fairly. Project maintainers
are obligated to respect the privacy and security of the reporter of any
incident.

For the full enforcement ladder (correction → warning → temporary ban →
permanent ban), see the
[Enforcement Guidelines section of the Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
